这是qemu向导工具
本工具可以用向导的形式来使用qemu创建或删除虚拟机, 启动虚拟机, 启动物理设备上的操作系统
运行"qemuguide"可以打开向导主界面
运行"qemuguidecreate"可以打开创建虚拟机向导
运行"qemubootexisting"可以打开启动已存在虚拟机的向导
运行"qemubootdisk"可以打开启动物理设备上的操作系统的向导
运行"qemudelete"可以删除以创建的虚拟机

里面的"installandcreate.mp4"是安装本软件和创建虚拟机的演示视频
"bootfromphysicaldevice.mp4"是启动物理设备的视频
